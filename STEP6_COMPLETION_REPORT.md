## 🔧 Ключевые технические решения

### Использование Set для selectedPayments
```javascript
state.selectedPayments = new Set()  // O(1) поиск по ID
```
**Преимущество:** Быстро проверить, выбран ли платеж: `state.selectedPayments.has(id)`

### Асинхронная история для каждого платежа
```javascript
for (const payment of state.payments) {
  if (ids.includes(payment.id)) {
    await addHistory({ ... });  // Один запрос на платеж
  }
}
```
**Преимущество:** Полная история всех операций, не блокирует UI

### Функция getPlural для русского языка
```javascript
getPlural(count, "выбран", "выбраны", "выбрано")
// 1 → "выбран", 2 → "выбраны", 5 → "выбрано", 21 → "выбран"
```
**Преимущество:** Естественный язык для пользователя
